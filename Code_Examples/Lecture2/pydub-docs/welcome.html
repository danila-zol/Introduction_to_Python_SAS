Welcome to pydub docs!

A powerful audio editing library for basic audio synthesis and music production (maybe?)

Why use pydub?

Pydub provides a handy toolset for making your very own music track and learn more advanced programming along the way! It is a partial successor to MUSIC-N programming languages in the text-based music programming and supports both editing existing raw audio on your system and synthesizing audio using built-in generators.
