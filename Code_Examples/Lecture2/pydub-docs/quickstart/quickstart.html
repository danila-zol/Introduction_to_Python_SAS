Quickstart

Opening a raw sound file using pydub

In pydub every sound is contained in a library-specific object called AudioSegment. You can create an AudioSegment by opening a file like this:

from pydub import AudioSegment

drum = AudioSegment.from_file("./drumloop.wav")

This will make python look for a file called drumloop.wav and read it into a variable drum of the pydub-specific type AudioSegment

Now that we have the sound on our hands we can perform various editing with very simple syntax!

# Make it loop (I mean, it says drum loop on the tin...)
drum_loop = drum * 5

# Make it 3db louder!
louder = drum_loop + 3

# Add a smooth fade in effect the end that lasts for 250ms
result = louder.fade_out(250)

Cool! Now let's play it using play function from the module pydub.playback:

from pydub.playback import play

play(result)

After we're done, we can export the results using export funciton:

# Audio format can be altered by specifying the format option
result.export("./drumloop_edit.mp3", format="mp3")
