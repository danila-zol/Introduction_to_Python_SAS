Basic sound synthesis

Generators

Pydub allows generating sound using its generators[1] from the module pydub.generators. To do that, just import the generator of your liking from the module and instantiate it in your program:

from pydub.generators import Sine

# Create a generator of frequency 400Hz
generator = Sine(400)

We've just imported a Sine wave generator and instantiated it in a generator variable with a frequency of 400 Hz

Next we need to create an AudioSegment using this generator

# Use this generator to synthesize a sound for 2 seconds (2000ms)
tone = generator.to_audio_segment(duration=2000)

Good! Now, let's play it!

from pydub.playback import play

play(tone)
